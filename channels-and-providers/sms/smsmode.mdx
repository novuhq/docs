You can use the [smsmode](https://www.smsmode.com/) provider to send sms to your customers using the Novu Platform with a single API.

# Getting Started

To use the smsmode provider in the sms channel, you will need to [create a smsmode account](https://ui.smsmode.com/register) for accessing your dashboard.

<Steps>
    <Step title="Access to dashboard">
        Connect to your [smsmode](https://ui.smsmode.com/home) dashboard.
    </Step>
    <Step title="Access to API keys section">
        Search for `Settings` in the side menu and then proceed to the `API keys` [section](https://ui.smsmode.com/settings/keys).
    </Step>
    <Step title="Generate your API key">
        Generate if needed an API key by clicking on `Generate a new API Key` button where you can set a name, and add IP address restriction depending on your needs.
    </Step>
    <Step title="Getting your API key">
        Copy and keep aside the API key you want from the list. Remember that your API key is a secret that should not be shared as it will enable to use API with your account.
    </Step>
    <Step title="Access to Senders IDs section">
        Search in the same `Settings` part of the side menu and then proceed to the `Senders IDs` [section](https://ui.smsmode.com/settings/senders).
    </Step>
    <Step title="Generate a sender ID">
        Generate if needed a sender ID by clicking on `+ Add` button where you can set the Sender ID that must be 3 to 8 alphanumerics characters with no special characters nor spaces.
    </Step>
    <Step title="Getting your sender ID">
        Copy and keep aside the Sender ID from your list. It will be the name that will be displayed on the SMS sender name.
    </Step>
</Steps>

# Creating a smsmode integration with Novu

- Visit the [Integrations store](https://web.novu.co/integrations) on the Novu web dashboard.
- Click on Add a Provider.
- Select smsmode service.
- Enter your smsmode API key you generated earlier.
- Fill in the `From` field with the `sender ID` you created earlier.
- Click on the `Disabled` button and mark it as `Active`.
- Click on the **Update** button.
- You should now be able to send notifications using smsmode in Novu.
