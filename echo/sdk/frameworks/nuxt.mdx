sidebarTitle: "Nuxt"
---

Echo works out of the box with Nuxt.js. You can use the `@novu/echo/nuxt` package to serve your Echo client in your Nuxt.js application.

For a boilerplate project with Vue Email, view: [Nuxt.js Boilerplate](https://github.com/novuhq/echo-nuxt-example/)

<CodeGroup>
```ts API Routes
import { serve } from "@novu/echo/nuxt";
import { echo } from "../echo";

export default defineEventHandler(serve({ client: echo }));
```
</CodeGroup>

Check out a comprehensive [guide on sending notifications in Nuxt with Vuemail](/guides/echo-guides/echo-nuxt-vuemail).
