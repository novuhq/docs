```bash
npx novu init --secret-key=<YOUR_NOVU_SECRET_KEY>
```

This will create a new Next.js application with Novu Framework pre-configured. The application will include:

- A sample workflow demonstrating the flexibility of Novu using Step Controls
- React Email integration for building beautiful email templates
- TypeScript configuration for end-to-end type safety
- Environment variables setup with your Novu Secret Key
