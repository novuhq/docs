The `Bell` component displays the notification bell icon. It can also be used to show the number of unread notifications.

<Callout type="warn" title="Breaking change">
  The `renderBell` prop now passes an `unreadCounts` object instead of a number. To access the total unread count, use `unreadCounts.total` and `unreadCounts.severity` to access unread counts for each severity level.
</Callout>

## Custom Bell with Inbox component

You can pass any custom components to the `renderBell` prop when using Inbox component. This gives you access to the full `unreadCounts` object, including severity-based counts.

```tsx
import { Inbox, SeverityLevelEnum } from '@novu/react';

function Novu() {
  return (
    <Inbox 
      applicationIdentifier="YOUR_APPLICATION_IDENTIFIER" 
      subscriber="YOUR_SUBSCRIBER_ID" 
      renderBell={(unreadCount) => {
        return (
          <div>
            <p className="bg-purple-300 p-4 inline-flex">
              New ${unreadCount.total} notifications
            </p>
            <p className="bg-purple-300 p-4 inline-flex">
              High severity $
              {unreadCount.severity[SeverityLevelEnum.HIGH]}{' '}
              notifications
            </p>
          </div>
        );
      }} 
    />
  );
}
```

## Custom Bell with Bell component

You can also directly customize the Bell component with the renderBell prop:

```tsx
import { Inbox, Bell } from '@novu/react';
import { BellIcon } from 'lucide-react';

function Novu() {
  return (
    <Inbox 
      applicationIdentifier="YOUR_APPLICATION_IDENTIFIER" 
      subscriber="YOUR_SUBSCRIBER_ID" 
    >
      <Bell
        renderBell={(unreadCounts) => (
          <div>
            <span>{unreadCounts.total}</span>
            <BellIcon />
          </div>
        )}
      />
    </Inbox>
  );
}
```

## Custom Popover Integration

`<Inbox />` can be mounted in your own popover component. For further customization, you can also use the `renderBell` and `renderNotification` render props.

Below is an example of how to use `<Inbox />` with [Radix UI](https://www.radix-ui.com/):

```tsx
import React from 'react';
import * as Popover from '@radix-ui/react-popover';
import { BellIcon, Cross2Icon } from '@radix-ui/react-icons';
import { Inbox, Bell, Notifications } from '@novu/react';
import './styles.css';

const PopoverDemo = () => (
  <Inbox applicationIdentifier="YOUR_APPLICATION_IDENTIFIER" subscriber="YOUR_SUBSCRIBER_ID">
    <Popover.Root>
      <Popover.Trigger asChild>
        <Bell
          renderBell={(unreadCounts) => (
            <div>
              <span>{unreadCounts.total}</span>
              <BellIcon />
            </div>
          )}
        />
      </Popover.Trigger>
      <Popover.Portal>
        <Popover.Content className="PopoverContent" sideOffset={5}>
          <Notifications />
          <Popover.Close className="PopoverClose" aria-label="Close">
            <Cross2Icon />
          </Popover.Close>
          <Popover.Arrow className="PopoverArrow" />
        </Popover.Content>
      </Popover.Portal>
    </Popover.Root>
  </Inbox>
);

export default PopoverDemo;
```
