## Layouts

The composition of the individual components can generate several popular inbox layouts.

### Inbox with bell (Default)

A trigger button, usually located at the top right corner of the screen, which triggers the popover component when clicked.

<Tabs items={['Default Bell', 'Custom Bell']}>
  <Tab>

```tsx
import { Inbox } from '@novu/react';

function Novu() {
  return <Inbox subscriber="1" applicationIdentifier="1" />;
}
```

  </Tab>

```tsx
import { Inbox, Bell } from '@novu/react';
import { BellIcon } from './icons';

function Novu() {
  return (
    <Inbox applicationIdentifier="YOUR_APPLICATION_IDENTIFIER" subscriber="YOUR_SUBSCRIBER_ID">
      <Bell
        renderBell={(unreadCount) => (
          <div>
            <span>{unreadCount}</span>
            <BellIcon />
          </div>

      />
    </Inbox>

}
```

### Side menu Inbox

To render a side menu Inbox, use the `<Notifications />` component as a direct child of the `<Inbox />` component. You can hide or show it using custom logic.

```tsx
import { Inbox, Notifications } from '@novu/react';

function Novu() {
  return (
    <Inbox applicationIdentifier="YOUR_APPLICATION_IDENTIFIER" subscriber="YOUR_SUBSCRIBER_ID">
      {showSideMenu && <Notifications />}
    </Inbox>
  );
}
```

### Full page Inbox

Similiar to the side menu Inbox, you can use the `<Notifications />` component as a direct child of the `<Inbox />` component. You can hide or show it using custom logic.

```tsx
import { Inbox, Notifications } from '@novu/react';

function Novu() {
  return (
    <Inbox applicationIdentifier="YOUR_APPLICATION_IDENTIFIER" subscriber="YOUR_SUBSCRIBER_ID">
      <Notifications />
    </Inbox>
  );
}
```

### Inbox with a custom popover

```tsx
import { Bell, Inbox, InboxContent } from '@novu/react';
import { Popover, PopoverContent, PopoverTrigger } from '@/components/ui/popover';

export default function CustomPopoverPage() {
  return (
    <Inbox applicationIdentifier="YOUR_APPLICATION_IDENTIFIER" subscriber="YOUR_SUBSCRIBER_ID">
      <Popover>
        <PopoverTrigger>
          <Bell />
        </PopoverTrigger>
        <PopoverContent className="h-[500px] w-[400px] overflow-auto p-0">
          <InboxContent />
        </PopoverContent>
      </Popover>
    </Inbox>
  );
}
```
