## Create an agent in the dashboard

1. Go to the [Agents page](https://dashboard.novu.co/agents) in the Novu dashboard.
2. In the sidebar, click **Agents**.
3. Click **Create agent**.
4. Fill in the required fields:
   - **Agent name**: Display name for your agent (used in Slack and Microsoft Teams when users mention the agent).
   - **Identifier**: Unique slug used in code. Auto-generated from the name; cannot be changed after creation.
   - **Description**: Optional summary of the agent's purpose.
5. Click **Set up agent**.

![Create a new agent](/images/agents/custom-code-agent/setup-your-agent/create-a-new-agent.gif)

After creation, Novu opens the guided setup page where you connect providers and copy the scaffold command for your bridge app.

## Connect a provider

On the agent setup page:

1. Open the **Select provider** dropdown.
2. Choose a provider (for example, **Slack**).
3. Follow the provider-specific steps shown in the dashboard.

For a full Slack walkthrough including app creation and install, see [Quickstart](/agents/custom-code-agent/quickstart).
